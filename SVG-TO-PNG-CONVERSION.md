# Инструкция по конвертации SVG иконок в PNG

## Созданные SVG файлы иконок мишени

Я создал для вас SVG иконки мишени в черно-лазурных цветах в следующих размерах:

- `target-icon.svg` - Оригинальный файл (512x512)
- `target-icon-16.svg` - 16x16 пикселей (для favicon)
- `target-icon-32.svg` - 32x32 пикселя (стандартный favicon)
- `target-icon-64.svg` - 64x64 пикселя
- `target-icon-128.svg` - 128x128 пикселей
- `target-icon-256.svg` - 256x256 пикселей
- `target-icon-512.svg` - 512x512 пикселей

## Способы конвертации в PNG

### 1. Онлайн конвертеры (рекомендуется)

**Рекомендуемые сайты:**
- [Convertio.co](https://convertio.co/svg-png/) - качественный и быстрый
- [SVG to PNG Online](https://svg2png.com/) - специализированный конвертер
- [CloudConvert](https://cloudconvert.com/svg-to-png) - надежный сервис

**Как использовать:**
1. Откройте один из рекомендованных сайтов
2. Загрузите SVG файл
3. Выберите размер (если необходимо)
4. Нажмите "Конвертировать" 
5. Скачайте PNG файл

### 2. Локальные программы

**Windows:**
- **Paint.NET** - бесплатный редактор с плагином для SVG
- **Adobe Illustrator** - профессиональный инструмент
- **Inkscape** - бесплатная альтернатива

**macOS:**
- **Preview** (встроенная программа) - откройте SVG и сохраните как PNG
- **Adobe Illustrator** - профессиональный инструмент
- **Sketch** - дизайнерский инструмент

**Linux:**
- **Inkscape** - полностью бесплатный
- **GIMP** - может открывать некоторые SVG файлы

### 3. Командная строка (продвинутый способ)

Если у вас установлен **ImageMagick**:

```bash
# Для каждого файла выполните:
convert target-icon.svg target-icon.png
convert target-icon-16.svg target-icon-16.png
convert target-icon-32.svg target-icon-32.png
# и так далее...
```

## Рекомендуемые размеры для разных целей

- **16x16** - Иконка в браузерной вкладке (favicon)
- **32x32** - Стандартный размер для большинства браузеров
- **64x64** - Хорошее качество для средних разрешений
- **128x128** - Для высокого качества на мобильных устройствах
- **256x256** - Для современных веб-приложений
- **512x512** - Максимальное качество для ретина-дисплеев

## Советы по использованию

1. **Всегда сохраняйте оригинальный SVG файл** - он масштабируется без потери качества
2. **Используйте разные размеры** для оптимального отображения на разных устройствах
3. **PNG файлы идеально подходят для**:
   - Фавикон веб-сайта
   - Иконок мобильных приложений
   - Социальных сетей
   - APK файлов

Все остальные файлы вашего приложения остались без изменений!
